Core Features

Everything your team needs

Built with real-world requirements in mind — the kind of features clients expect in production.

Kanban Board

Drag & drop tasks across fully customizable status columns. Each project defines its own workflow.

  • Custom status columns per project
  • Drag-drop reorder of columns
  • Instant optimistic UI updates

Role-Based Access Control

Granular permissions enforced on both frontend and backend. Three role levels with distinct capabilities.

  • Admin · Manager · Member
  • JWT stateless authentication
  • Frontend UI adapts per role

Smart Dashboard

Role-aware statistics surface the data each user actually needs — not a one-size-fits-all view.

  • Completion percentages
  • Overdue task detection
  • Recent project activity

Full Audit Trail

Every project and task change is recorded — who changed what, when, and the before/after values.

  • Project activity log
  • Task status change history
  • Reassignment tracking

Overdue Detection

Business logic computed server-side: tasks past their due date that aren't done are flagged automatically.

  • Server-side business rule
  • Visual overdue indicators
  • Filtered in dashboard stats

Soft Delete

Nothing is permanently erased. Projects and tasks are soft-deleted with a timestamp, preserving data integrity.

  • is_deleted + deleted_at columns
  • Recoverable data model
  • Audit-safe architecture
